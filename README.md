# WakeUpDarling-

Parts I used:

Arduino Uno

Pushbuttons (2)

LCD 16x2

Wires

Resistors (220Ω, 1kΩ)

Potentiometer

Coffee Machine

Diode (SOD83-12)

Transistor NPN (SOT37)

Relay SPDT 5V (coil) 120VAC

Soldering iron

Grounding Converters (2) like the one in the picture
